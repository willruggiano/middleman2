Branch-head decoration in the commit picker

A small icon marks any commit that is also the tip of another local branch. Hover the icon to see the branch name(s). Hover rows to see active/hover states.

Placement A — right gutter (recommended)

Marker sits with the date, in the calm right gutter. Message text stays left-aligned across every row, so the list scans cleanly; branch info reads as metadata alongside the date.

Placement B — right after the SHA
